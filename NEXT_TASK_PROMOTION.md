## Задача 1: Виральный призыв (ВЫПОЛНИТЬ ПЕРВЫМ)

### Что менять

#### `posts-pool.ts`

1. Добавить поле `viralCta?: string` в тип `Post`
2. В каждый пост добавить подходящий по рубрике призыв (см. ниже)
3. В `formatPost()` дописать CTA в конец

```typescript
// Добавить в тип Post:
viralCta?: string;

// Изменить formatPost:
export function formatPost(post: Post): string {
  const base = `${post.emoji} *${post.title}*\n\n${post.body}`;
  const cta = post.viralCta
    ? `\n\n${post.viralCta}`
    : "\n\nСохрани себе. Пригодится.";
  return base + cta;
}
```

**CTA по рубрикам:**
- 🧼 Гигиена: `"Покажи это кому-нибудь из близких — пригодится раньше, чем кажется."`
- 🎣 Разбор: `"Если кто-то из твоих недавно чуть не попался — перешли. Без комментариев."`
- 🛠 Инструмент: `"Сохрани и кинь другу, который думает, что с ним такого не случится."`
- 🧪 Проверь себя: `"Скинь кому-нибудь и проверьте вместе — интереснее не одному."`
- 📖 История: `"Перешли тому, кто знает похожую историю. Иногда это буквально помогает."`

## Задача 5: Мониторинг-агент площадок

### Алгоритм

1. Раз в сутки (через cron в `scheduler.ts`) запускать `monitoring-agent.ts`
2. Отправлять запрос в Perplexity API:
   ```
   Найди 3–5 свежих обсуждений интернет-мошенничества за последние 24 часа 
   на русскоязычных платформах (Пикабу, ВКонтакте, форумы).
   Для каждого: тема, ссылка (если есть), краткое описание.
   ```
3. Парсить ответ → формировать список топ-3
4. Отправлять автору в Telegram как личное сообщение

### Новый файл: `lib/monitoring-agent.ts`

```typescript
export async function runMonitoringAgent(): Promise<void> {
```
